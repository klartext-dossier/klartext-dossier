# Symbols

## LaTeX special symbols

* '&' Ampersand
* '%' Percent
* '{' Curly brace open
* '}' Curly brace close

## Checkboxes symbols

* '☑' Checked checkbox
* '☐' Unchecked checkbox

## Whitespace

* ' ' Thin space

## Others

* '±' Plus minus sign
* '°' Celsius

## Card symbols

* '♠' Spades
* '♥' Hearts
* '♦' Diamonds
* '♣' Clubs
* '⚠' Alert
